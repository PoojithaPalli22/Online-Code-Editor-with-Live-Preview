# Online Code Editor with Live Preview

## Project Description
The Online Code Editor with Live Preview is a web-based application that allows users to write and edit HTML, CSS, and JavaScript code directly in the browser and instantly see the output in a live preview window. This project helps developers and students test and learn web development without installing any software.

## Features
- Write HTML, CSS, and JavaScript in the editor
- Live preview of the output in real time
- Simple and user-friendly interface
- Instant code execution in the browser
- Useful for learning and practicing web development

## Technologies Used
- HTML
- CSS
- JavaScript
